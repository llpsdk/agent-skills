# Agent Skills
This agent skill provides guidance and troubleshooting for integrating the Large Language Platform SDK into your agentic application.

Be sure to register at [Large Language Platform](https://llphq.com) before integrating with our SDK.

## Getting Started
Install this skill via `npx skills add llpsdk/agent-skills`

Using a coding assistant of your choice, you can prompt it to integrate the SDK.

### Sample Prompt
> I want to integrate the llpsdk into my application. Perform all necessary steps to install and connect to the
> Large Language Platform.

> [!NOTE]
> We currently only support Javascript, Python, and Go.
